# 📈 Stock & Crypto Visualizer

A responsive web application built with Flask, Chart.js, and Yahoo Finance that lets users view historical stock and cryptocurrency price charts. Includes zoom, pan, and interactive crosshair tracking for an enhanced trading-platform-like experience.

---

## 🚀 Features

- 🔍 Search & visualize price charts for stocks and cryptocurrencies
- 🕒 Select time ranges: 5 days, 1 month, 3 months, 6 months, 1 year
- 📊 Interactive Chart.js line chart with:
  - Zoom in/out (mouse wheel or pinch)
  - Pan (CTRL + drag)
  - Crosshair pointer with axis tracking
- 💰 Displays latest price
- 📱 Fully responsive and mobile-friendly UI

---

## 📦 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript, Chart.js, chartjs-plugin-zoom
- **Backend:** Python, Flask
- **Data Source:** Yahoo Finance (via `yfinance`)
