# Car Parts Management System

A full-stack CRUD web application built with PHP, MySQL, JavaScript, and Bootstrap.
This project demonstrates RESTful API integration, dynamic UI updates, and
client-side data management using jQuery and DataTables.

---

## Features

- RESTful PHP API
- Full CRUD operations (Create, Read, Update, Delete)
- Dynamic form auto-population
- Searchable and paginated DataTable
- Modal window for detailed item views
- Responsive Bootstrap UI
- User Details page (UI only)

## Run Locally

This project is designed to run locally.

1. Install XAMPP / MAMP / WAMP
2. Import `car_parts.sql` into MySQL
3. Place the project folder in `htdocs`
4. Update database credentials in `database.php`
5. Open `http://localhost/car-parts-management-system`
